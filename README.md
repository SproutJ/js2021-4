const app = express(); // 서버 생성
app.use((request, response) => {
    response.send('<h1>Hello express</h1>');
}); // request 이벤트 리스너 설정

    console.log('Server running at http://127.0.0.1:52273');

```

    - 서버가 제공함.

    2XX 성공  
    3XX 리다이렉트
    4XX 클라이언트 오류
    5XX 서버 오류
    ```
    - 상태 코드를 지정할 때는 status() 메소드 사용.
- 리다이렉트(3XX, 특수한 상태 코드)

```

console.log(foo);
```

if(foo.indexOf('아침') >= 0) {
    console.log('좋은 아침입니다.');
}
else {
    console.log('안녕히 주무세요.');
}
```

```
